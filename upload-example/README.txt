This is an example upload directory structure.

Place your BIDS-formatted neuroimaging data here before running the upload command.

For BIDS format requirements, see: https://bids.neuroimaging.io/

Example structure:
  dataset_description.json
  participants.tsv
  sub-01/
    anat/
      sub-01_T1w.nii.gz
    func/
      sub-01_task-rest_bold.nii.gz
